command mode
================

:set nu
----------------
替檔案設行數

:set ic/noic
----------------
在檔案內搜尋時開啟/關閉大小寫敏感

:set hlsearch/nohlsearch
---------------
在檔案搜尋時開啟/關閉高亮顯示

gg
---------------
到檔案頁首

GG
---------------
到檔案頁尾

數字+G(EX：10G)
---------------
到指定行數

yy
---------------
複製該行

dd
---------------
刪除該行

數字+xx(EX：10xx)
---------------
刪除該行10個字元

pp
---------------
貼上複製的東西
